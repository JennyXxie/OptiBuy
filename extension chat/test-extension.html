⚠️ Next Steps Required

To make the extension fully functional:

1. Load Extension in Chrome

Go to chrome://extensions/ → Enable "Developer mode" → "Load unpacked" → Select your OptiBuy folder

2. Get API Keys

Gemini API (Required): https://makersuite.google.com/app/apikey

SerpAPI (Optional): https://serpapi.com/

3. Configure API Keys

Click extension icon → Settings (⚙️) → Add your API keys → Save
