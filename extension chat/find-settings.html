🔧 How to Find API Key Settings in OptiBuy DealBot

🚨 You're in the Wrong Place!

The Chrome extension settings page you're seeing does NOT have API key fields.

You need to access the extension's internal settings panel, not Chrome's system settings.

📍 Step 1: Find the Extension Icon

Look for the OptiBuy DealBot icon in your Chrome toolbar:

  • 🔍 Check the top-right corner of Chrome
  • 🔍 Look for a puzzle piece icon (extensions menu)
  • 🔍 The OptiBuy DealBot icon should be visible

If you don't see it: Click the puzzle piece → Pin OptiBuy DealBot

📍 Step 2: Click the Extension Icon

Click the OptiBuy DealBot icon (not right-click, just click)

This should open a popup window with:

  • 💬 Chat interface
  • 🎯 Quick action buttons
  • ⚙️ Settings button in the footer

📍 Step 3: Find the Settings Button

In the popup, look at the bottom footer for:

  • ⚙️ Settings button (gear icon)
  • 🗑️ Clear chat button (trash icon)

Click the settings button (⚙️)

✅ What You Should See in Settings Panel:

  • Gemini API Key field: "Enter your Gemini API key"
  • SerpAPI Key field: "Enter your SerpAPI key"
  • Save Settings button

📍 Step 4: Add Your API Keys

Get Gemini API Key:

  1. Go to: https://makersuite.google.com/app/apikey
  2. Sign in with Google
  3. Click "Create API Key"
  4. Copy the key (starts with AIza...)

Add to Extension:

  1. Paste Gemini API key in first field
  2. Click "Save Settings"
  3. Status should change to "Online"

🔍 Troubleshooting

If you don't see the popup when clicking the icon:

  • ✅ Make sure extension is enabled in chrome://extensions/
  • ✅ Try refreshing the extension
  • ✅ Check if popup is blocked by Chrome

If you don't see the settings button in the popup:

  • ✅ Look at the very bottom of the popup
  • ✅ The settings button is a gear icon (⚙️)
  • ✅ It should be in the footer area
