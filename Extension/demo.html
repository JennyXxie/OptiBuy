🛍️ Shopping Price Tracker Chrome Extension

A powerful Chrome extension that helps users track product prices across multiple e-commerce platforms including Amazon, Temu, and Shein.

🎯 Key Features

  • Slot-filling Dialog: Step-by-step product search criteria collection
  • Multi-platform Search: Amazon, Temu, and Shein integration
  • Smart Filtering: By rating, platform, and description needs
  • Price Tracking: Automatic monitoring with notifications
  • Dual Output: JSON + natural language results

📋 Slot-Filling Process

Step 1: Product Name (Required)
User enters: "iPhone 15"
Step 2: Rating Filter (Optional)
Options: No preference, ≥4.5★, ≥4.0★, ≥3.5★
Step 3: Source Filter (Optional)
Multi-select: Amazon, Temu, Shein, All supported
Step 4: Description Need (Optional)
Options: None, Concise overview, Detailed spec summary

  • Stop tracking from notifications

🚀 Installation Instructions

  1. Open Chrome and navigate to chrome://extensions/
  2. Enable "Developer mode" in the top right
  3. Click "Load unpacked" and select this extension folder
  4. Click the extension icon in your Chrome toolbar
  5. Start searching for products!
